# olympic-analyses

## Init Data Analyses API

```sh
cd data_analyses

#Optional
python3 -m venv venv
# For linux
source venv/bin/activate
# For windows
venv\Scripts\activate

# Mandatory
pip install -r requirements.txt
python3 api/run.py
```

## Init Data Vis

```sh

cd data_vis

npm install
npm run dev
```

## Docker Command

```sh
docker-compose up --build
```
